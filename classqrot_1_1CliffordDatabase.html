std::size_t qrot::CliffordDatabase::GetTMove ( std::size_t  idx) const

Get the index of T-moved gate.

Let C be the gate of idx. Calculate C' such that the equation C * T = P * T * C' holds true. if idx < 64, P = I else if idx < 128, P = H else P = SH

Return the index of C'
