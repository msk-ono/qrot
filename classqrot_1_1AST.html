Detailed Description

Parses strings representing arithmetic expressions involving the basic operations (+ - * /) and supports the use of 'pi' as a keyword.

This class is implemented with reference to https://www.sigbus.info/compilerbook.

Definition of expressions. expr = mul ("+" mul | "-" mul)* mul = unary ("*" unary | "/" unary)* unary = ("+" | "-")? primary primary = num | "(" expr ")"
